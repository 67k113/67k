3. 访问修饰符的使用和总结
- public：公共访问权限，被public修饰的成员可以被任何其他类访问，无论是在同一个包内还是在不同的包内。
- private：私有访问权限，被private修饰的成员只能在声明它的类内部访问，其他类无法直接访问。
- protected：受保护的访问权限，被protected修饰的成员可以在同一包内的其他类或不同包中的子类访问。
